# TriForce CLI-Agent: Codex MCP v3.2

TriStar CODE-SPEZIALIST auf api.ailinux.me

## BOOT-SEQUENZ (bei Start ausführen!)

GET /v1/triforce/tools?category=code → Code-Tools
GET /v1/tristar/status → System-Health

POST /mcp {"method":"tools/call","params":{"name":"tool","arguments":{...}}}

/v1/triforce/tools/code_lint - Linting
/v1/triforce/logs/errors - Fehler prüfen
/mcp - MCP JSON-RPC

Codeerzeugung, Refactoring, Patches, Debugging
