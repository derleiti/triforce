# AILinux Dark Theme (CSS++ v1.0)

This directory contains the official AILinux Dark Theme, built using the CSS++ Design Language.

## Files

- **`ailinux-dark.csspp`**: The source code. Edit this to change the theme.
- **`ailinux-dark.css`**: The compiled CSS. **Do not edit this manually.**

## Integration

To use this theme in WordPress:

1.  **Compile:**
    ```bash
    npm run csspp:compile csspp/themes/ailinux-dark.csspp > assets/css/ailinux-dark.css
    ```

2.  **Enqueue:**
    Add this to your `functions.php`:
    ```php
    wp_enqueue_style('ailinux-dark-theme', get_template_directory_uri() . '/assets/css/ailinux-dark.css', [], '1.0.0');
    ```

3.  **Usage in HTML:**
    Apply classes based on the material names defined in the `.csspp` file.
    ```html
    <div class="material-card">
        <div class="material-stack">
             <h1>Hello World</h1>
             <button class="material-button-primary">Click Me</button>
        </div>
    </div>
    ```

## Features
- **Neon Glows**: Automatic hover states generated by `lighting: hover-glow()`.
- **Volumetric Depth**: Glassmorphism and fabric textures via `volumetrics`.
- **Animations**: Built-in `fade-in`, `pulse` via the compiler runtime.
