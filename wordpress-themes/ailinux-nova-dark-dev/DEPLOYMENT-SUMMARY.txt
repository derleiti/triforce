================================================================================
DEPLOYMENT METHODS
================================================================================

METHOD 1: IN-PLACE ACTIVATION (EASIEST)
  Theme is already installed in WordPress.

  Steps:
  1. WordPress Admin → Appearance → Themes
  2. Click "Activate" on "AILinux Nova Dark"
  3. Go to Customizer → CSS++ Theme → Select theme
  4. Publish

  OR use WP-CLI:
  ./ACTIVATE.sh

METHOD 2: ZIP DEPLOYMENT
  Create ZIP:
    cd /home/zombie/wordpress/html/wp-content/themes/
    zip -r ailinux-nova-dark.zip ailinux-nova-dark-dev \
      -x "*/node_modules/*" "*/.git/*" "*/tests/*"

  Upload:
    Admin → Appearance → Themes → Add New → Upload
    Select ZIP → Install → Activate

METHOD 3: GIT DEPLOYMENT
  Clone to new server:
    git clone <repo> /path/to/themes/ailinux-nova-dark-dev
    cd ailinux-nova-dark-dev
    npm install --production
    npm run build:production
